# POO-Atelier 3 : Gestion d’un parc de voitures

## Description
Ce projet est un simple gestionnaire de parc de voitures en Python. Il utilise des classes pour stocker les informations des voitures et pour gérer une liste de voitures.

## Fichiers du projet
- `voiture.py` : classe Voiture
- `parc_voitures.py` : classe ParcVoitures
- `main.py` : programme avec un menu
- `test_parc.py` : version simple pour tester le programme
- `.gitignore` : fichiers à ignorer par Git

## Comment utiliser
1. Ouvrir un terminal dans le dossier du projet
2. Lancer le programme :
```bash
python main.py
```
3. Suivre les instructions du menu

## Menu disponible
1. Ajouter une voiture
2. Retirer une voiture
3. Afficher toutes les voitures
4. Chercher une voiture
5. Voir le nombre de voitures
6. Quitter

## Exemple de voiture
- Plaque : AB123
- Marque : Toyota
- Modèle : Camry
- Propriétaire : Jean

## Commande de test
Pour vérifier le fonctionnement du programme, exécuter :
```bash
python test_parc.py
```
